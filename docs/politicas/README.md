# Políticas de la Empresa

Esta carpeta contiene los archivos PDF de las políticas de GDS INFRAESTRUCTURA.

## Archivos requeridos:

1. **politica-ambiental.pdf** - Política Ambiental
2. **politica-calidad.pdf** - Política de Calidad  
3. **politica-seguridad-salud.pdf** - Política de Seguridad y Salud en el Trabajo
4. **politica-igualdad-equidad.pdf** - Política de Igualdad y Equidad

## Instrucciones:

1. Coloca los archivos PDF en esta carpeta (`docs/politicas/`)
2. Asegúrate de que los nombres de archivo coincidan exactamente con los mencionados arriba
3. Los archivos serán descargables desde el footer de la página web

## Nota:

Los enlaces en el footer están configurados para descargar estos archivos automáticamente cuando el usuario hace clic en ellos.
